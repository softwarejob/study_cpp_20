#include <iostream>
使程式能夠將資料輸出到螢幕上
是一個預處理指令給c++預處理器的訊息，編譯器在編譯程式之前呼叫．
<iostream>的內容．此標頭是包含編譯器在編譯任何輸出程式時所需的資訊檔案．
空白-常被編譯器忽略
int main(){
是每一個c++程式的一部分．main後的括號表示他是一個函式．
c++程式通常由一個或是多個函式合類組成．
每個程式中的一個函式必須命名為main，這就是c++程式開始執行的地方．
關鍵字 int表示在main完成執行後，它返回一個整數（整數）值．關鍵字和return保留給c++使用
}
std::cout該物件通常連線到螢幕
\n Newline將螢幕光標定位到下一行的開頭
\t Horizontal tab.將螢幕光標向右移動到下一個選項卡停止
\r Carriage return.將螢幕光標定位到當前行的開頭;不前進到下一行
\a Alert.敲響系統鈴聲
\\ Backslash.包含字串中的反斜線字元
\' Single quote.包含字串中的單引號字元
\" Double quote.在字串中包含一個雙引號字元
